## 📦 About wechat-sdk-qclaw

wechat-sdk-qclaw is a simple tool that helps you run a WeChat chatbot on your Windows PC. It works like a personal assistant using WeChat. You can use it to automate replies and manage your chats without any coding skills. The program copies the style of ItChat, making the experience familiar for anyone who uses WeChat.

This app runs on Windows and works with your personal WeChat account to automate simple tasks. It’s built to be easy to use even if you don’t know programming.

---

## 🔍 What can you do with it?

- Send automatic replies to your contacts.
- Manage personal chats without being online all the time.
- Receive and respond to messages while the app is running.
- Customize your chatting routine with basic settings.
- Use it as a base for more advanced chatbot projects (for users with coding knowledge).

## 🔐 Logging in to WeChat

This app connects to your personal WeChat, so you need to log in.

1. When the app opens, it will show a QR code.
2. Open WeChat on your phone.
3. Use the WeChat **Scan** option to scan the QR code on your computer screen.
4. Confirm the login on your phone if asked.

Once logged in, the app will sync your chat data and start running the assistant.

## 🛠️ Basic Configuration

To update how the chatbot works:

1. In the program folder, find the file named `config.json` or `settings.ini`.
2. Open this file with a simple text editor like Notepad.
3. Change values such as:
   - Auto-reply message text.
   - Which chats to target (groups or individual contacts).
   - Time delay for replies.
4. Save the file and restart the app to use the new settings.
